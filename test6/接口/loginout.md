#### 用例： 登出
- 权限：学生/老师已经登录。
- 功能： 已经登录的用户登出平台。
- API请求地址： 接口基本地址/v1/api/logout
- 请求方式 ： post
- 请求实例：

```
  {   
  "student_no":"201510414211"
  }

```
- 请求参数说明：

	数名称	| 说明
---|---
student_no|已经登录用户的student_no值

- 返回实例：
```
{         
      "status": true,
      "info": "登出成功"
  }
```


参数名称 | 说明
---|---
status | bool类型，true表示正确的返回，false表示有错误
info | 返回结果说明信息
